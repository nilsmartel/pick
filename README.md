# pick

Pick a specific line from stdin or a file

```
pick 0.1.0
Pick a specific line from stdin or a file

USAGE:
    pick [OPTIONS] <line-number>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -f, --file <input>    optional input file, defaults to stdin

ARGS:
    <line-number>
```
